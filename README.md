# 🧮 Python Tkinter Calculator

A simple calculator application built using Python and Tkinter. This project provides a clean graphical user interface (GUI) for performing basic arithmetic calculations.

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Decimal number support
- Clear button
- Error handling for invalid expressions
- User-friendly interface

## 🛠️ Technologies Used

- Python 3
- Tkinter (Python GUI Library)

## 📸 Preview

The calculator provides:
- Display screen for input and results
- Numeric buttons (0-9)
- Arithmetic operators
- Clear button
- Equal button for calculations
